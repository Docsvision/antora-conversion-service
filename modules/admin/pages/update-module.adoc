[#conditions]
== Условия обновления

* Установка новой версии должна выполняться в нерабочее время.
* Перед установкой обновления ознакомьтесь с xref:ROOT:requirements.adoc[требованиями] для работы модуля.
* Для корректной работы данной версии сервиса требуется определённая версия модуля {wc}, требования указаны в пункте "xref:ROOT:requirements.adoc[]".

Режим обновления предназначен для установки новой версии сервиса без удаления существующей версии.
