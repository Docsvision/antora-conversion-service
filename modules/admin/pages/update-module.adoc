[#conditions]
== Условия обновления

* Установка новой версии должна выполняться в нерабочее время.
* Перед установкой обновления ознакомьтесь с требованиями к xref:webclient:ROOT:requirements-dv.adoc[системе] и xref:webclient:ROOT:requirements-software.adoc[программному обеспечению].
* Для корректной работы данной версии сервиса требуется определённая версия модулей {pl}, {bo} и {wc}, требования указаны в пункте "xref:ROOT:requirements.adoc[]".

Режим обновления предназначен для установки новой версии сервиса без удаления существующей версии.
